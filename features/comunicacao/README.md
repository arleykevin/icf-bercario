# features/comunicacao

**Comunicação & Alertas** (Fase 1). Comunicados com botão **"Ciente"** (registro imutável de
leitura), preferências de notificação e **Web Push** (payload genérico, sem PII na tela de
bloqueio). Chat 1:1 com quiet hours entra na v2.

Princípio: reduzir ruído — canal crítico/saúde nunca silenciável; rotina vai em digest.
Ver PLANO.md §3 e §5.2.

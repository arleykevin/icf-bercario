# features/seguranca

**Segurança & Gestão** (Fase 1). Perfil da criança (saúde, alergias), **autorizados a
retirar** (documento **cifrado** — Vault/pgsodium — nunca em texto puro, acesso auditado),
**check-in/check-out** com prova de custódia (registro imutável) e, na v1.1, autorização de
medicamento assinada.

Regra: dado de documento é minimizado e cifrado; toda leitura sensível é auditada.
Ver PLANO.md §4.2 e §5.
